



    general
ctrl+o = open file
ctrl+k , ctrl+o = open folder
ctrl+Shift+P = Show Command Palette
ctrl+R = recent file
ctrl+w = close tab
ctrl+shift+t = reopen tab

    text editing
ctrl+L = select the entire line
ctrl+u = cut the entire line
ctrl+c = copy selection
ctrl+z = undo
ctrl+y = redo
shift+click = draggable selection

alt+ ↑|↓ = Move line up/down
ctrl+enter = insert line below
ctrl+shift+enter insert line above
ctrl+]|[ = Indent/outdent line
ctrl+Home = Go to beginning of file
ctrl+End = Go to end of file
ctrl+shif+k = toggle line Comment
alt+z = toggle word wrap 
alt+shift+left = expand both sides

    search and replace
ctrl+F = Find
ctrl+H = Replace
ctrl+D = select next and next and next match

    Multi-cursor 
ctrl+click = Insert cursor
ctrl+shift+down|top = add cursor
crtl+shift+i Insert cursor at end of each line selected after text being selected

    view layout toggle
alt+shift+b Toggle Sidebar visibility
alt+shift+s Toggle Status bar visibility
alt+shift+p Toggle Panel visibility
